# HCL Training
This repository consists Core Java based design of Banking Transactions. As part of our HCL Tech training, this is a assessment which tests our ability to use the core java skills and design a operations done by ATM machine by using abstract classes and methods

## What is Abstract Classes?
An abstract classes are the regular classes which is a blue print of an object.
These abstract classes don't allow you to write the logic and implementations. 
The Abstract Classes consists only abstract methods definitions.
